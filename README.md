# 🩺 Médicos & Dentistas

O projeto **Médicos & Dentistas** é uma plataforma voltada para o setor de saúde, desenvolvida para conectar profissionais de saúde e pacientes de forma eficiente. A aplicação foca em uma interface intuitiva, performance otimizada e uma arquitetura de código limpa e escalável.

## 📺 Demonstração e Deploy

  * **Link do Projeto (Deploy)**: [Vite Project](https://www.google.com/search?q=https://vite-project-six-flame.vercel.app/)

-----

## 🧠 O que aprendi de mais valioso

Nesta jornada de desenvolvimento, consolidei conhecimentos críticos em engenharia de software frontend:

  * **Componentização Estratégica**: Desenvolvi uma biblioteca de UI própria (Buttons, Cards, Inputs) focada em reusabilidade e consistência visual em toda a plataforma.
  * **Estilização com Escopo**: Utilizei **SASS Modules** para garantir que a estilização fosse modular, isolada e livre de efeitos colaterais globais.
  * **Navegação SPA**: Implementei rotas dinâmicas utilizando **React Router**, garantindo transições fluidas e uma experiência de usuário sem recarregamentos de página.
  * **Modern Tooling**: Adotei o **Vite** como ferramenta de build, aproveitando um ambiente de desenvolvimento ultra-rápido e otimizado.

-----

## 🚀 Funcionalidades

  * **Hero Section**: Apresentação principal com foco em conversão e usabilidade.
  * **Missão e Impacto**: Seções dedicadas a detalhar o propósito social e os números da plataforma.
  * **Área do Voluntário**: Fluxo completo para engajamento de novos profissionais de saúde.
  * **Formulários Inteligentes**: Componentes de input customizados para coleta de dados estruturada.
  * **Design Responsivo**: Adaptabilidade total para diferentes tamanhos de tela (Desktop, Tablet e Mobile).

-----

## 🛠️ Tecnologias Utilizadas

  * **React 18**: Base da aplicação para construção de interfaces declarativas.
  * **Vite**: Ferramenta de build e servidor de desenvolvimento de nova geração.
  * **SASS (SCSS) Modules**: Estilização avançada com variáveis, mixins e escopo local.
  * **React Router Dom**: Gerenciamento de rotas e navegação.
  * **React Icons**: Integração de ícones vetoriais de alta qualidade.

## 🔧 Instalação e Execução

1.  **Clone o repositório**:
    ```bash
    git clone https://github.com/thaisny/m-dicos-dentistas---desafio---vnw.git
    ```
2.  **Entre na pasta do projeto**:
    ```bash
    cd m-dicos-dentistas---desafio---vnw/vite-project
    ```
3.  **Instale as dependências**:
    ```bash
    npm install
    ```
4.  **Inicie o servidor local**:
    ```bash
    npm run dev
    ```
